# 日程块

日程块是在思源文档中插入的独立挂件块，用于提供日历块和周历块两种日程视图。

## 功能

- 支持日视图和周视图
- 日历块与周历块在块内严格分开，插入后不再在块内切换视图
- 左上角显示当前绑定日期或 ISO 周，点击可跳回绑定日/周，齿轮可修改绑定值
- 支持从顶栏菜单、命令面板和斜杠菜单插入日历块或周历块
- 支持在时间轴上拖选时间段新建日程
- 支持点击日程后编辑标题、时间、全天、颜色和备注
- 支持拖动日程调整时间，拖拽边缘修改时长
- 支持用不同颜色区分日程

## 定位

日程块不是完整的日历系统，也不是数据库日历视图的替代品。数据库日历视图更适合管理带有日期属性的块、任务或资料条目；现有日历/日程类插件通常更强调全局日历、日记跳转、节假日信息或任务管理。

本插件解决的是另一类痛点：在任意思源文档中直接嵌入一个固定绑定日期或周次的轻量日程块。例如在项目文档、会议记录、课程安排、读书计划或个人工作台中，直接放置当天或本周的时间轴，不需要先建立数据库，也不需要离开当前文档去打开全局日历。

## 用法

启用插件后，点击顶栏「日程块」图标，选择「插入日历块」或「插入周历块」即可在当前光标位置附近插入。

也可以在文档中输入 `/日历块` 或 `/周历块` 后选择对应块。

插入后，直接在块内操作：

- 点击左上角日期或周号，回到这个块绑定的日期或周
- 点击齿轮，修改这个块绑定的日期或周
- 在时间轴上拖选一段时间，新建日程
- 点击空白时间点，快速创建一小时日程
- 点击已有日程，编辑或删除
- 拖动已有日程，调整开始和结束时间
- 使用颜色选项区分不同类型日程

## 数据

当前版本的日程数据由挂件统一保存到思源工作空间：

`/data/storage/schedule-block/events.json`

日程块自身会保存视图类型和绑定日期，用于下次打开时恢复状态。
