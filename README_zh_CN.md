# 日程块

日程块是在思源文档中直接嵌入的轻量时间轴挂件，用于在项目文档、会议记录、课程安排、读书计划或个人工作台里安排某一天或某一周的日程。

它不是全局日历，也不是数据库日历视图的替代品。它关注的是一个更窄、更顺手的场景：不用建数据库，不用离开当前文档，就能在笔记里放一个可编辑、可拖拽、可长期保存的日程块。

## 核心功能

- 支持日历块和周历块，两类块插入后保持独立，不在块内混切视图。
- 每个块绑定一个日期或 ISO 周次，顶部锚点按钮可快速跳回绑定日期/周次。
- 插入新块时会自动从文档标题识别日期或周次；识别不到时使用当前日期。
- 设置按钮可修改绑定日期/周次，并设置默认事件时长。
- 新建日程默认 30 分钟，可在块设置中改为 15、30、45 或 60 分钟。
- 新建日程块默认高度可完整显示 24 小时，不需要手动拉伸才能看到全天。
- 支持拖选时间段新建日程；空白处单击不会新建，双击才按默认事件时长新建。
- 点击已有日程可编辑标题、日期、时间、全天状态、颜色和备注。
- 支持拖动日程调整开始时间，拖拽边缘修改时长。
- 鼠标悬停在带备注的日程上会显示备注浮层，移开后自动隐藏。
- 支持 `Cmd/Ctrl+Z` 撤销上一条日程数据操作。

## 交互细节

- 拖选、双击或工具栏「＋」新建后，输入内容会自动保存。
- 点击弹窗外部会保留已自动保存的结果。
- 点击「取消」或按 `Esc` 会撤回本次编辑：新建时删除草稿，编辑时回滚到打开弹窗前的内容。
- 跨天事件会明确显示起止日期，例如 `6月12日 23:00` 到 `6月13日 00:00`，避免午夜边界被误解。
- 短日程使用紧凑单行显示，事件高度严格按真实时长渲染，不会遮挡相邻日程。

## 颜色与筛选

- 日程可使用一组 Google Calendar 风格颜色区分类型。
- 工具栏设置按钮右侧提供颜色筛选按钮，可选择当前日程块需要显示哪些颜色的事件。
- 筛选色板顺序与新建/编辑日程时的颜色顺序一致。
- 全选颜色时显示全部日程；此时点击某个颜色会直接切换为“只看这个颜色”。
- 筛选状态下可继续点击其它颜色来增减显示范围；也可以点击「全部」一键恢复全部颜色。

## 日期头与天气

日期头采用两排极简布局：

- 第一排显示公历日期与农历日期。
- 第二排显示星期几与天气图标。

天气支持四种常用状态：晴、阴、雨、雪。点击天气图标即可选择或清除当天的天气。未设置天气时显示一个中性的空天气徽标，不暗示任何具体天气。

天气数据独立保存，不会影响日程事件。

## 用法

启用插件后，可以通过以下方式插入：

- 顶栏「日程块」菜单。
- 命令面板。
- 文档中的斜杠菜单：`/日历块` 或 `/周历块`。

插入后直接在块内操作即可：

- 文档标题含 `2026-6-13`、`2026/06/13`、`2026年6月13日`、`20260613`、`2026-W26` 或 `2026年第26周` 时，新块会自动绑定到对应日期或周次。
- 点击锚点按钮回到绑定日期或绑定周次。
- 点击齿轮修改绑定日期/周次和默认事件时长。
- 点击漏斗按钮按颜色筛选日程。
- 设置、筛选、新建和天气按钮都支持再次点击同一按钮关闭弹窗。
- 拖选时间段新建日程。
- 双击空白时间点快速新建默认时长日程。
- 点击已有日程编辑或删除。
- 悬停在带备注的日程上快速查看备注。
- 拖动已有日程调整时间。
- 点击日期头天气图标设置当天状态。

## 数据存储

日程数据统一保存到思源工作空间：

`/data/storage/schedule-block/events.json`

天气数据独立保存到：

`/data/storage/schedule-block/weather.json`

日程块自身还会在块属性中保存视图类型、绑定日期/周次、默认事件时长等状态，用于下次打开时恢复。

## 适用场景

- 日记、周计划、个人工作台。
- 项目文档中的阶段安排。
- 会议记录中的当天议程。
- 课程表、读书计划、写作计划。
- 任何需要把时间轴直接放在笔记内容里的场景。
